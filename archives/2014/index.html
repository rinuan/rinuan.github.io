Hexo 安裝除錯

$ npm install hexo -g 安裝之錯誤訊息

解決方式：

改用 $ sudo npm install hexo -g 安裝

$ hexo generate 錯誤訊息

解決方式：

檢查 _config.yml 的格式是否有誤，冒號後必須加上空格
